# Employee Leave Management System

## Project Overview
Employee Leave Management System is a web-based application developed using Java technologies to manage employee leave requests efficiently.

This system allows employees to apply for leave and allows admin to approve or reject leave requests through an interactive dashboard.

---

## Technologies Used

- Java
- Servlet
- JSP
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse IDE

---

## Features

### Admin Module
- Admin Login
- Dashboard View
- Add Employee
- Update Employee
- Delete Employee
- Manage Departments
- View Leave Requests
- Approve Leave
- Reject Leave
- Admin Settings

### Employee Module
- Employee Login
- Apply Leave
- View Leave Status
- View Leave History

---

## Database
MySQL Database used for storing:

- Employee Details
- Department Details
- Leave Records
- Admin Credentials

## How to Run

1. Import project into Eclipse IDE
2. Configure Apache Tomcat Server
3. Create MySQL database
4. Update database credentials in DBConnection.java
5. Run project on server
6. Open browser and access application
